## 一、先把今天定性清楚

今天很痛，但它不是“你完了”，也不是“你不适合交易”。它是一笔非常贵的学费，暴露的是一个清晰问题：

```text
你定义了企稳条件，
但条件没有成立时，
你没有执行降仓，
反而把账户推到接近满仓。
```

今天亏损的表面原因是科技大跌，深层原因是仓位在错误时间放大。科创50下跌 7.70%，创业板下跌 5.71%，中际旭创、中芯国际、寒武纪都没有企稳，这种盘面下，账户不应该接近 100% 股票仓位。

所以今天不是能力归零，而是系统报警：

- 方向判断可以错；
- 买点可以错；
- 但仓位不能失控；
- 亏损可以接受；
- 但不能用更大的仓位去对抗已经失效的判断。

这件事最重要的结论是：

```text
你要修的不是自尊。
你要修的是仓位开关。
```

## 二、今晚只做三件事

今晚不要研究新题材，不要翻小作文，不要试图找一个“明天反包”的理由让自己舒服。今晚只做三件事。

### 1. 承认亏损已经发生

今天账户回撤已经发生，不能靠后悔撤销。越反复想“如果早上卖了”“如果不满仓”“如果不买新仓”，情绪会越陷越深。

正确动作是把亏损从情绪里拿出来，放进系统里：

```text
亏损金额：-29,079.38
亏损原因：锚点未企稳时满仓
修复动作：明天先降仓，不补仓
```

只要它被记录、被拆解、被转化成规则，它就不再只是伤口，而是系统升级的材料。

### 2. 停止复仇交易

今晚最危险的念头是：

- 明天一定要赚回来；
- 明天开盘就加仓做反包；
- 今天跌这么多，明天肯定修复；
- 我不能接受账户跌破 50 万附近；
- 我必须证明自己没错。

这些念头都不是交易计划，是受伤后的反击冲动。明天不能带着这个状态交易。

今晚给自己立一条硬规则：

```text
明天不以赚钱为目标。
明天只以恢复可控为目标。
```

### 3. 睡前写一句话

睡前只写一句话：

```text
我今天亏惨了，但我没有被这一笔交易定义。
```

你需要把自己和账户分开。账户今天受伤了，但你这个人没有被市场判死刑。

## 三、明天的唯一目标

明天只有一个目标：

```text
把账户从失控满仓，恢复到可操作状态。
```

不是赚钱，不是抄底，不是证明超节点对，也不是把今天亏的赚回来。

### 明天的仓位目标

| 盘面状态 | 仓位目标 | 动作 |
|---|---:|---|
| 强修复 | 最高 5-6 成 | 也要降仓，保留最强 |
| 弱修复 | 4-5 成 | 留强去弱 |
| 继续杀跌 | 3 成以内 | 先活下来 |

### 明天的处理顺序

| 顺序 | 动作 | 原因 |
|---|---|---|
| 1 | 先处理海光信息 | 最大亏损源，拖累账户心态 |
| 2 | 再判断盛科通信-U | 超节点弹性锚，必须看能否强修复 |
| 3 | 再看华丰、锐捷、亨通 | 今日新仓，明天接受市场验证 |
| 4 | 不补亏损票 | 禁止用加仓解决情绪问题 |
| 5 | 不开新方向 | 先恢复现金和主动权 |

明天你只要把仓位从 99.7% 降下来，就是胜利。哪怕明天卖完后它反弹，也不是失败。因为你现在最缺的不是收益，而是主动权。

## 四、三天恢复计划

### 第一天：止血

时间：2026-07-03。

目标：

```text
仓位降到可控。
不补仓。
不新开仓。
不因为反弹后悔。
```

允许做的事：

- 卖弱票；
- 留最强的 1-2 只观察；
- 把现金恢复到至少 40%-50%；
- 收盘后只复盘执行，不评判自己。

禁止做的事：

- 开盘后立即加仓；
- 因为某只票快速反弹而追进去；
- 想用一笔交易赚回 29,000；
- 盘中频繁切换方向。

### 第二天：冷却

时间：下一个交易日。

目标：

```text
降低交易频率。
只观察市场是否真正企稳。
```

观察清单：

- 科创50是否站回关键位置；
- 中际旭创是否企稳；
- 寒武纪或中芯国际是否企稳；
- 海光信息是否止跌；
- 超节点是否有核心锚重新走强。

当天最多只允许一笔新交易，且仓位不超过 1 成。

### 第三天：重建

目标：

```text
用小仓位重新找回节奏。
```

重建不是满仓反击，而是重新证明自己能按规则做事。你要找回的是执行力，不是刺激感。

重建仓位规则：

| 状态 | 单票仓位 |
|---|---:|
| 刚经历大亏后的前三天 | 不超过 1 成 |
| 连续两天执行计划 | 可提高到 1.5 成 |
| 连续三天没有冲动交易 | 可恢复正常试错仓 |

## 五、如何停止越亏越急

大亏后最可怕的不是亏损本身，而是你会变成另一个人：

- 看见反弹就想追；
- 看见别人赚钱就焦虑；
- 看见亏损就想补；
- 看见账户数字就心跳加速；
- 明知道不该动，还是想动。

这不是你意志力差，而是亏损触发了身体的应激反应。解决办法不是骂自己，而是设置动作隔离。

### 1. 盘中只允许看三样东西

明天盘中只看：

```text
指数锚：科创50
核心锚：中际旭创、寒武纪、中芯国际
账户动作：仓位是否下降
```

其他信息全部降级。小作文、群消息、涨幅榜、别人盈利截图，都先别看。

### 2. 每次下单前停 30 秒

下单前问自己三个问题：

| 问题 | 如果答案是否定 |
|---|---|
| 这是计划内动作吗 | 不下单 |
| 这是为了降风险吗 | 不下单 |
| 这笔交易会让我更冷静吗 | 不下单 |

大亏后的交易，第一标准不是能不能赚钱，而是会不会让你更失控。

### 3. 今天不评价自己的人格

你可以说：

- 今天仓位错了；
- 今天执行错了；
- 今天没有遵守企稳条件；
- 今天应该降仓。

但不要说：

- 我废了；
- 我不适合交易；
- 我永远做不好；
- 我就是韭菜。

这些话没有复盘价值，只会继续伤害执行力。

## 六、这次亏损真正教你的东西

这次亏损不是来告诉你“别做科技”，也不是来告诉你“超节点不行”。它真正教你四件事。

### 1. 条件没有成立，不能交易升级

你自己定义了企稳条件：

```text
中际旭创企稳
中芯国际或寒武纪企稳
```

条件没有成立，仓位就不能升级。以后只要自己提前写下条件，就必须尊重它。

### 2. 满仓必须是奖励，不是赌注

满仓只能出现在：

- 指数确认；
- 核心锚确认；
- 主线扩散确认；
- 自己已有利润垫；
- 账户心态稳定。

今天这些条件都不满足，所以满仓变成了赌注。

### 3. 同一风险因子不是分散

盛科通信-U、海光信息、华丰科技、锐捷网络、亨通光电名字不同，但风险因子高度一致：

```text
AI硬件
科创成长
超节点
通信网络
光通信
```

这不是五只票分散，而是一条主线满仓。

### 4. 能走出来的人不是不亏，而是亏后不乱

真正的进步不是以后不亏，而是亏损之后仍然能：

- 按计划减仓；
- 不复仇；
- 不补弱票；
- 不立刻追新题材；
- 把错误变成规则。

## 七、账户重建规则

从明天开始，账户进入“修复模式”。修复模式下，规则比收益更重要。

### 1. 仓位上限

| 阶段 | 仓位上限 |
|---|---:|
| 明天 | 5 成 |
| 如果继续弱 | 3 成 |
| 连续两天执行纪律 | 5 成 |
| 重新出现强主线并有利润垫 | 6-7 成 |
| 禁止状态 | 无利润垫满仓 |

### 2. 单票上限

| 类型 | 单票上限 |
|---|---:|
| 修复期观察票 | 0.5-1 成 |
| 修复期核心票 | 1-1.5 成 |
| 正常试错票 | 1-2 成 |
| 情绪失控期 | 不新增 |

### 3. 三个禁止

```text
禁止补亏损票。
禁止开盘前十分钟冲动追。
禁止因为想回本而加仓。
```

### 4. 三个允许

```text
允许卖飞。
允许空仓。
允许慢慢赚回来。
```

## 九、下一次交易前检查表

以后每次想重仓前，必须回答下面问题：

| 问题 | 必须满足 |
|---|---|
| 指数是否支持 | 是 |
| 核心锚是否企稳 | 是 |
| 板块是否扩散 | 是 |
| 是否已有利润垫 | 是 |
| 是否有明确止损线 | 是 |
| 是否不是为了回本 | 是 |
| 单票仓位是否可承受 | 是 |
| 如果明天低开 5%，我是否能执行 | 是 |

只要有三个以上答案不是“是”，就不能重仓。

最后，把今天写成一句系统规则：

```text
当我定义的企稳条件没有成立时，
我只能降仓或等待，
不能加仓，
更不能满仓。
```

你会走出来。不是靠明天大涨走出来，而是靠明天开始重新按规则做事走出来。
