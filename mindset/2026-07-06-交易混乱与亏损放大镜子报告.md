## 一、一句话结论

你最近的交易不是输在没有研究、没有逻辑、没有热点感知，而是输在三个更底层的问题：

```text
计划能写出来，但盘中执行会被情绪覆盖；
知道谁弱，但弱票经常占大仓；
明白要等锚点确认，但市场没确认时仍会把仓位打满。
```

这不是简单的“手法乱”，而是交易系统里最危险的一类问题：**认知和执行断裂**。

你已经能说出正确原则，也能整理产业链、锚点和炒作顺序；但当账户出现亏损、踏空、焦虑、想快速修复时，实际下单会偏离这些原则。亏损持续放大，不是因为某一只股票害你，而是因为你在错误的市场状态里放大了仓位，在弱票上停留太久，在强弱已经分出来后没有果断切换。

## 二、最近亏损放大的真实链条

### 1. 从主动低仓到失控满仓

你的资金曲线里，最典型的一段是：

| 日期 | 状态 | 账户含义 |
|---|---|---|
| 2026-06-26 | 仓位 1.3% | 主动权很强，几乎空仓等待 |
| 2026-06-29 | 仓位 11.2% | 小仓试错，风险可控 |
| 2026-06-30 | 真实股票仓位约 28.7% | 科技修复，参与合理，但可用资金过低 |
| 2026-07-01 | 真实股票仓位约 55.0% | 科创开始分歧，仓位已经偏高 |
| 2026-07-02 | 股票仓位约 99.7% | 锚点未企稳，账户满仓同风险因子 |
| 2026-07-06 | 仓位 99.5% | 再次接近满仓，亨通光电单票约 38.1% |

问题不在于仓位高本身，而在于仓位高的时候，市场并没有给出足够确认。

### 2. 锚点没有企稳，但你选择赌修复

2026-07-02 早盘，你自己已经定义了科技企稳的两个标志：

```text
1. 中际旭创企稳；
2. 中芯国际或者寒武纪企稳。
```

收盘结果是三个锚点都没有企稳，科创50 大跌，AI硬件和国产算力核心继续杀跌。但账户却被推到接近股票满仓。

这说明你不是不知道怎么看盘，而是在盘面给出否定答案后，仍然希望市场按你的主观逻辑走。

## 三、你现在最核心的八个问题

### 1. 你会把“逻辑正确”误当成“现在可以买”

你研究产业链很勤快，能快速找到核心环节、扩散路径、相关标的。但你的弱点是：一旦逻辑看起来顺，就容易忽略盘面是否已经确认。

典型表现：

- 海外链有逻辑，就容易忽视海外链当下是否真的强；
- 超节点有想象力，就容易忽视锚点是否破位；
- 光纤光缆有业绩预期，就容易忽视个股是否弱于同题材核心；
- 半导体有中报线，就容易把后排也当成核心。

镜子结论：

```text
逻辑只能让一只股票进入观察池；
只有盘面强弱，才决定它能不能上仓位。
```

### 2. 你对“满仓”的危险感不够稳定

你知道满仓危险，也写过很多次要降仓。但一到盘中，满仓又会反复出现。

满仓对你最大的伤害不是一天亏多少，而是让你失去主动权：

- 想买强票，没钱；
- 想换仓，必须先卖；
- 想止损，又怕卖飞；
- 想低吸，只能补弱票；
- 遇到低开，情绪直接失控。

镜子结论：

```text
你不是不能满仓，而是不能在锚点未确认、账户无利润垫、持仓强弱混乱时满仓。
```

### 3. 你经常把“分散持仓”做成“同风险因子集中”

持有五只票，看起来分散；但如果它们都属于 AI硬件、科创、超节点、通信、光通信、半导体扩散，那么本质上是同一个风险因子。

2026-07-02 的组合就是典型例子：

```text
盛科通信-U
海光信息
华丰科技
锐捷网络
亨通光电
```

它们名字不同，但都吃科技高波动水位。科创50和AI硬件一杀，组合会一起下跌。

镜子结论：

```text
真正的分散，不是股票数量多；
真正的分散，是风险因子不共振。
```

### 4. 你止损太依赖“明天可能修复”

亏损票刚跌时，你能看出弱；但真要卖时，会被几个念头拖住：

- 它逻辑还在；
- 跌多了明天可能反包；
- 我卖了它就涨怎么办；
- 这票是核心受益；
- 先等等看。

问题是，短线交易里，“等等看”经常会把小亏拖成大亏。

镜子结论：

```text
你的止损不是不会设，而是到了止损点后不愿意承认判断已经失效。
```

### 5. 你容易用新题材缓解旧亏损焦虑

亏损扩大后，你会更想找下一个方向：机器人、医药、海外链、存储、超节点、AI上游材料。研究新方向本身没问题，但如果它是为了缓解旧仓位的痛苦，就会变成复仇交易的入口。

典型链条是：

```text
旧仓亏损
-> 情绪不舒服
-> 找新题材
-> 觉得新题材有机会
-> 卖不干净旧弱仓，又开新仓
-> 仓位更满，组合更乱
```

镜子结论：

```text
新机会不能用来治疗旧亏损。
旧仓位没处理干净之前，新机会只能小仓验证。
```

### 6. 你盘前计划很清楚，盘中执行太容易变形

你的文档里经常有非常明确的条件：

- 10:30 前不修复就降仓；
- 锚点不企稳不加仓；
- 弱票反弹减；
- 不补亏损票；
- 总仓位降到 60%-70%。

但真正的问题是盘中没有把这些条件当成“命令”，而是当成“建议”。于是市场一波拉升、一个小作文、一个反抽，就会改变动作。

镜子结论：

```text
你现在最缺的不是更多观点，而是把自己写下的规则变成不可讨价还价的执行。
```

### 7. 你有时过于追求“赚回来”

大亏之后，你最危险的念头不是悲观，而是急着修复账户。

急着赚回来，会导致：

- 仓位恢复过快；
- 买点降低标准；
- 弱反弹误判为反转；
- 对亏损票补仓；
- 对新题材过度兴奋。

镜子结论：

```text
亏损后第一目标不是赚钱，是恢复可控。
```

### 8. 你对“强”和“弱”的处理不够残酷

你能识别强弱，但对弱票下手不够快，对强票给仓不够果断。

这会造成很难受的结构：

```text
强票涨了，但仓位小，赚不多；
弱票跌了，但仓位大，亏很多；
最后账户感觉一直被拖着走。
```

镜子结论：

```text
交易账户不是平均主义。
强票应该得到仓位奖励，弱票必须被仓位惩罚。
```

## 四、这些问题背后的心理模式

### 1. 你害怕错过，所以容易提前上车

看到题材发酵、别人赚钱、核心票高开，你会担心“再不买就没了”。于是会在确认不足时提前提高仓位。

本质是踏空焦虑。

修正方式：

```text
宁愿错过第一段，也不要在没有确认时吃系统性回撤。
```

### 2. 你害怕承认错，所以容易拖弱票

卖出亏损票，意味着承认这笔交易错了。你会希望它先反弹一点，让自己体面退出。

但市场不会负责给你体面。

修正方式：

```text
止损不是否定自己，止损只是承认这笔交易的条件失效。
```

### 3. 你把研究能力当成交易安全感

你研究得越多，越容易觉得自己“有依据”。但交易不是论文答辩，市场不按你的研究深度给收益。

修正方式：

```text
研究负责建立观察池；
交易负责执行强弱和仓位。
两者不能混在一起。
```

### 4. 你容易把账户波动和自我评价绑定

亏损放大时，你会怀疑自己是不是不适合交易；盈利时，又容易想加快速度证明自己。

这会让仓位跟情绪走，而不是跟市场状态走。

修正方式：

```text
账户只是反馈系统，不是人格评分系统。
```

## 五、最需要立刻停止的行为

### 1. 停止在锚点未企稳时满仓

硬规则：

```text
中际旭创、寒武纪、中芯国际、科创50这些核心锚没有企稳时，
科技类总仓位不得超过 50%。
如果当天指数弱、锚点弱、持仓弱，科技仓位不得超过 30%。
```

### 2. 停止让单只弱票超过 20%

硬规则：

```text
亏损超过 5% 的单票，仓位不得继续超过 20%。
亏损超过 8% 的单票，必须降到观察仓或重新定义中线计划。
短线账户里，不允许亏损票成为第一大仓。
```

### 3. 停止补亏损票摊成本

硬规则：

```text
没有板块共振、没有核心锚修复、没有个股重新强于板块，不补亏损票。
```

### 4. 停止同时买多个同风险因子扩散票

硬规则：

```text
同一条 AI硬件/超节点/通信/半导体链，最多保留 2-3 只。
其中必须有核心锚，不能全是扩散票。
```

### 5. 停止用新题材掩盖旧仓位问题

硬规则：

```text
旧仓位没有降到计划区间前，新题材只能观察或小仓试错。
不能一边弱票大仓，一边继续开新方向。
```

## 六、未来十个交易日的硬规则

### 1. 总仓位规则

| 市场状态 | 总仓位上限 |
|---|---:|
| 指数弱，核心锚弱 | 30% |
| 弱修复第一天 | 40% |
| 修复延续但未主升 | 50%-60% |
| 核心锚确认主升 | 70% |
| 刚经历大亏后的前三天 | 50% |

未来十个交易日，除非核心锚和持仓同时强，否则不要再接近满仓。

### 2. 单票仓位规则

| 股票状态 | 单票上限 |
|---|---:|
| 试错票 | 5%-10% |
| 有利润垫但非核心 | 10%-15% |
| 核心强票 | 15%-25% |
| 亏损超过 5% | 20% 以下 |
| 亏损超过 8% | 10% 以下或退出短线仓 |

### 3. 买入规则

每笔买入前必须回答：

```text
1. 它是不是当前资金选择的方向？
2. 它是不是这个方向里最强的一批？
3. 如果买错，止损线在哪里？
4. 买入后账户总仓位是否仍然可控？
5. 它和已有持仓是不是同一风险因子？
```

五个问题里有两个答不清楚，不买。

### 4. 卖出规则

卖出不再等“完全走坏”。未来十个交易日按这套顺序处理：

```text
先卖最大亏损源；
再卖弱于板块的票；
再卖没有核心地位的扩散票；
最后才处理强票利润。
```

### 5. 盘中规则

盘中不允许临时推翻盘前计划，除非出现新的市场事实。

新的市场事实必须是：

- 核心锚放量突破；
- 指数放量反包；
- 板块前排集体共振；
- 持仓明显强于板块；
- 原风险条件解除。

小作文、群里观点、别人赚钱截图、盘中一根拉升，都不算新的市场事实。

## 七、每天开盘前的镜子提问

每天开盘前问自己十个问题：

1. 今天大盘状态允许我进攻吗？
2. 科创50、创业板、上证的方向一致吗？
3. 中际旭创、寒武纪、中芯国际至少有两个企稳吗？
4. 我手里最大仓位是不是最强票？
5. 我手里最大亏损票是不是仓位过大？
6. 今天如果低开，我有没有现金和主动权？
7. 我现在想买，是因为计划，还是因为害怕错过？
8. 我现在不想卖，是因为强，还是因为不愿承认错？
9. 新买的票和旧持仓是不是同一风险因子？
10. 如果今天只能做一件事，是降风险，还是进攻？

如果第 4、5、6 个问题答案不好，今天优先降风险。

## 八、真正的改进方向

### 1. 少一点“找题材”，多一点“管仓位”

你不是缺题材。你现在项目里已经有足够多题材：

- 存储；
- 海外链；
- 超节点；
- 机器人；
- 医药；
- AI上游材料；
- 半导体设备；
- 先进封装；
- 光纤光缆。

真正缺的是：哪个题材确认强时，给多少仓；哪个题材不确认时，最多试多少仓；哪个题材走弱时，什么时候走。

### 2. 少一点“证明自己”，多一点“保护账户”

亏损后最重要的不是把钱马上赚回来，而是停止继续亏。

未来三天，只要能做到：

```text
不满仓；
不补弱票；
不让亏损票成为第一大仓；
不在锚点未企稳时加仓；
盈利票保护利润；
```

这就是进步。

### 3. 少一点“平均处理”，多一点“强弱排序”

每天收盘后必须给持仓排队：

```text
第一名：最强且最该留；
第二名：有利润垫，可继续观察；
第三名：中性，不加仓；
第四名：弱于板块，准备减；
第五名：最大风险源，优先处理。
```

交易不是把每只票都照顾好，而是让资金流向最值得留的位置。

### 4. 少一点“盘中临时发挥”，多一点“预案执行”

盘前写好的条件，盘中必须执行。

如果盘前写的是：

```text
10:30 前不能修复就减仓。
```

那到 10:30 就执行，不再临时解释。

如果盘前写的是：

```text
核心锚不企稳不加仓。
```

那核心锚不企稳时，就不能因为某只后排拉升去追。

## 九、最后的提醒

你最近亏损持续放大，最刺眼的缺点不是“不会选股”，而是：

```text
你经常知道正确答案，
但在情绪最强的时候没有执行正确答案。
```

所以这份报告不是让你否定自己，而是让你认清一个事实：

**你真正要练的不是更聪明，而是更稳定。**

接下来十个交易日，不要追求把亏损一次赚回来。你只做一件事：

```text
让账户重新回到可控状态。
```

可控的账户，才有资格谈收益；失控的账户，所有逻辑都会变成情绪的借口。

最后一句话：

```text
把强票当强票，把弱票当弱票；
把计划当命令，把仓位当生命线。
```
