# 2026-06-23 英伟达高温液冷新技术与 A 股炒作顺序

> 生成时间：2026-06-23  
> 用途：梳理英伟达 Rubin 代高温液冷参考设计的产业含义、A 股核心标的和短线炒作顺序。  
> 结论属性：交易研究，不构成投资建议；盘中以板块强度、成交量和核心股承接为准。

## 一、核心结论

英伟达这次液冷新技术的重点不是“浸没式液冷”，而是 **Rubin 代参考设计中的高温 Direct-to-Chip 冷板液冷**。

核心变化有三点：

1. **冷却位置前移**：热量直接在芯片端被冷板捕获，而不是主要依赖机房空调把热空气带走。
2. **冷却液温度提高**：液体回路可在约 45°C，也就是 113°F 附近运行，降低对低温冷冻水和传统冷机的依赖。
3. **排热方式变化**：通过高温液体回路 + 室外干冷器排热，目标是减少甚至接近消除传统冷却塔带来的数据中心用水。

交易上，这条线应按以下逻辑理解：

```text
Rubin 功耗提升
→ 单机柜热密度继续上升
→ 风冷/传统冷冻水方案压力加大
→ 高温冷板液冷成为主线方案
→ 冷板、CDU、管路快接、泵阀、换热器、干冷器、冷却液、密封材料受益
```

最正宗方向不是泛液冷，而是：

**冷板液冷 + CDU + 高温水路 + 干冷器 + 低导电/防泄漏冷却液 + 密封快接材料。**

## 二、这次英伟达新技术到底是什么

### 1. 事件

2026-06-22，外媒报道英伟达在 London Climate Week 期间披露下一代 AI 基础设施冷却方案，指向 Rubin generation reference design。公开报道提到：

- Rubin 代数据中心参考设计转向 fully liquid-cooled systems。
- 液体可在约 113°F / 45°C 运行。
- 热量在芯片端被直接捕获，并通过高温液体回路输送到室外干冷器。
- 目标是减少传统冷却塔用水，降低对传统制冷设备的依赖。

### 2. 技术本质

这不是简单说“液冷比例提升”，而是 **液冷温度区间和数据中心排热架构改变**。

传统数据中心常见路径：

```text
芯片发热
→ 风扇带走热空气
→ 机房空调/冷冻水系统降温
→ 冷却塔或冷机排热
```

Rubin 高温冷板液冷路径：

```text
芯片 / GPU / CPU / NVSwitch
→ 冷板直接取热
→ 高温冷却液循环
→ CDU/换热器
→ 室外干冷器排热
→ 减少冷却塔和冷冻水依赖
```

### 3. 和此前 Rubin 液冷判断的关系

本地已有结论仍然成立：

- Rubin 当前主线是 **Direct-to-Chip 冷板式液冷**。
- 浸没式液冷不是本轮最正宗主线。
- 本次新增重点在于 **高温运行、干冷器、水资源约束、冷却液安全性**。

所以炒作维度从原来的“Rubin 要用液冷”升级为：

**Rubin 要用更高温度、更少水、更高可靠性的整套液冷基础设施。**

## 三、产业链拆解

### 1. 芯片最近端：冷板

冷板仍是最核心、最直接的环节。高功耗 GPU、CPU、NVSwitch 等热源需要冷板直接贴合取热。

关注点：

- 冷板换热效率。
- 微通道设计。
- 低流阻。
- 封装贴合能力。
- 防泄漏可靠性。
- 大批量制造一致性。

对应 A 股：

- 银轮股份
- 飞荣达
- 金富科技
- 高澜股份
- 同飞股份

### 2. 机柜侧：CDU 与液冷分配

CDU 负责服务器侧冷却液和设施侧水系统之间的换热、控温、控压和监测。

本次高温液冷会提高 CDU、换热器、泵阀、传感器、控制系统的重要性。

对应 A 股：

- 英维克
- 申菱环境
- 高澜股份
- 同飞股份
- 曙光数创

### 3. 管路、快接头、分液歧管、密封

Rubin 级别机柜价值高，液冷系统最怕泄漏。高温液体循环下，对管路、快接、密封、防腐、监测要求更高。

对应 A 股：

- 科创新源
- 川环科技
- 沃尔核材
- 飞荣达
- 金富科技

### 4. 冷却液

Axios 报道提到英伟达方案中的循环液体类似水和丙二醇混合液。Tom's Hardware 此前报道 Supermicro 展示 Vera Rubin NVL72 机柜时，也强调新型冷却液电阻抗较传统冷却液大幅提升，核心诉求是降低轻微泄漏造成停机或损坏的风险。

A 股映射难度比冷板、CDU 更高，因为需要确认具体冷却液配方、客户认证和收入占比。

可观察方向：

- 冷却液材料。
- 防腐添加剂。
- 绝缘/低导电液体。
- 数据中心液冷专用工质。

可跟踪公司：

- 巨化股份
- 永和股份
- 三美股份
- 新宙邦
- 沃尔核材

注意：冷却液在 A 股里更容易被资金当作“新分支”短炒，但正宗度要弱于冷板和 CDU，必须等公告、订单或客户验证。

### 5. 干冷器与机房侧排热

本次新技术的边际变化之一，是从冷却塔/传统冷机向室外干冷器排热切换。干冷器、换热器、泵组、控制系统、水系统工程都会被重新定价。

对应 A 股：

- 申菱环境
- 英维克
- 佳力图
- 依米康
- 银轮股份
- 冰轮环境
- 汉钟精机

但这里要区分：

- **正向受益**：液冷集成、干冷器、换热、泵阀、控制系统。
- **可能被压制**：传统冷机、冷却塔、低温冷冻水依赖度较高的纯设备逻辑。

| 层级 | 方向 | 核心标的 | 弹性标的 | 交易定位 |

## 五、核心个股逻辑

### 英维克

- **定位**：液冷/CDU/数据中心温控中军。
- **逻辑**：AI 数据中心热密度提升，液冷渗透率提高，CDU 和机房热管理需求提升。
- **本次新技术映射**：高温液冷仍需要 CDU、监测、控压、换热和机房侧系统集成。
- **交易属性**：容量中军，适合看板块强弱，不是最高弹性。
- **验证点**：液冷收入占比、CDU 订单、AI 数据中心客户。

### 申菱环境

- **定位**：数据中心温控、液冷机房、特种空调。
- **逻辑**：高温液冷 + 干冷器排热会提高机房侧工程和系统集成价值。
- **本次新技术映射**：从冷却塔/冷机逻辑向干冷器、换热和液冷机房方案扩展。
- **交易属性**：中军偏弹性。
- **验证点**：数据中心液冷项目、干冷/换热订单、客户结构。

### 银轮股份

- **定位**：热管理、冷板、换热器。
- **逻辑**：从汽车热管理向数据中心液冷冷板和换热扩展，制造能力和热交换经验有价值。
- **本次新技术映射**：冷板 + 换热器 + 高温水路，贴合度高。
- **交易属性**：制造型中军，弹性取决于数据中心订单确认。
- **验证点**：AI 服务器冷板订单、数据中心收入占比。

### 飞荣达

- **定位**：导热材料、液冷板、服务器散热模组。
- **逻辑**：从导热材料向服务器散热模组和液冷板延伸。
- **本次新技术映射**：冷板、导热材料、结构件、密封防护都有映射。
- **交易属性**：中军偏弹性。
- **验证点**：液冷板订单、北美/国内服务器客户进展。

### 金富科技

- **定位**：液冷冷板、管路、AI 服务器液冷弹性票。
- **逻辑**：市场对其液冷子公司、产能扩张和客户链有较强预期。
- **本次新技术映射**：冷板、管路、快接、液冷模组都可被资金映射。
- **交易属性**：短线弹性强，容易成为情绪先锋。
- **验证点**：订单、产能、客户确认；若只有情绪拉升，冲高回落风险高。

### 高澜股份

- **定位**：液冷设备、CDU、电力电子冷却。
- **逻辑**：有液冷设备基础，AI 数据中心液冷渗透提升带来设备需求。
- **本次新技术映射**：CDU、高温液冷控制、换热设备。
- **交易属性**：弹性较高。
- **验证点**：数据中心液冷订单、CDU 产品、客户结构。

### 同飞股份

- **定位**：工业温控、液冷、数据中心温控。
- **逻辑**：工业温控能力向数据中心液冷延伸。
- **本次新技术映射**：机柜液冷设备、温控系统。
- **交易属性**：中高弹性。
- **验证点**：AI 数据中心订单、液冷收入占比。

### 科创新源

- **定位**：密封材料、液冷管路、热管理材料。
- **逻辑**：高温液体循环和高价值机柜提高密封、防泄漏、管路材料重要性。
- **本次新技术映射**：管路、密封、防泄漏细分环节。
- **交易属性**：细分弹性票。
- **验证点**：是否进入主流服务器液冷供应链。

### 川环科技

- **定位**：液冷管路、软管、汽车/服务器热管理。
- **逻辑**：冷板液冷需要大量可靠管路，管路系统可作为后排扩散。
- **本次新技术映射**：高温液路、软管、连接系统。
- **交易属性**：补涨弹性。
- **验证点**：服务器液冷管路订单和客户认证。

### 曙光数创

- **定位**：液冷数据中心、北交所高弹性。
- **逻辑**：市场辨识度高，容易被资金同时映射到冷板和浸没式。
- **本次新技术映射**：液冷数据中心系统。
- **交易属性**：波动大，适合作为情绪温度计。
- **风险点**：Rubin 当前主线不是浸没式，不能把浸没式当最正宗逻辑。

## 六、炒作顺序

### 第一阶段：消息发酵

资金先找辨识度最高的液冷中军和弹性票。

优先观察：

- 英维克
- 申菱环境
- 金富科技
- 高澜股份
- 飞荣达

盘面特征：

- 液冷板块高开。
- 中军有成交额。
- 弹性票快速拉升。
- 新闻关键词集中在“英伟达、Rubin、45°C、水冷、干冷器、节水”。

操作含义：

- 不追没有辨识度的后排。
- 先看中军是否放量承接。
- 若只有小票一字，中军不动，按情绪题材处理。

### 第二阶段：正宗分支确认

市场会从“液冷概念”切到更细环节：

```text
冷板
→ CDU
→ 管路/快接/密封
→ 干冷器/换热
→ 冷却液
```

最应该确认的是：

- 冷板：银轮股份、飞荣达、金富科技。
- CDU：英维克、申菱环境、高澜股份。
- 管路密封：科创新源、川环科技、沃尔核材。

操作含义：

- 只有冷板/CDU 中军继续强，液冷才可能从短线消息变成主线扩散。
- 如果冷却液、浸没式、边缘后排先乱拉，反而说明资金在做题材扩散末端。

### 第三阶段：产业链扩散

如果液冷得到确认，资金会扩散到：

- 干冷器。
- 换热器。
- 泵阀。
- 传感器。
- 冷却液。
- 导热材料。
- 数据中心总包和机房侧工程。

对应观察：

- 申菱环境、英维克、佳力图、依米康。
- 银轮股份、冰轮环境、汉钟精机。
- 巨化股份、永和股份、三美股份、新宙邦。
- 飞荣达、中石科技、思泉新材。

操作含义：

- 扩散阶段只能做强者，不做被动跟涨。
- 越往后排，越需要降低仓位和预期。
- 冷却液如果没有公告或客户验证，只能按题材弹性处理。

### 第四阶段：高位分歧

液冷题材常见风险是高开一致、盘中分化。

风险信号：

- 英维克、申菱环境不强，只有小票冲高。
- 金富科技、高澜股份高开后放量回落。
- CPO、半导体、存储同步走弱，AI 硬件主线不支持扩散。
- 大盘缩量，涨停减少，后排炸板。

操作含义：

- 后排先走。
- 弹性票不强就降。
- 中军若放量滞涨，不追。
- 只有冷板/CDU 核心承接住，才考虑二次分歧低吸。

## 七、盘中验证清单

### 判断题材真强

同时满足 3 条以上，说明液冷线有主线扩散资格：

1. 英维克、申菱环境至少一只放量强势。
2. 金富科技、高澜股份、飞荣达至少两只强于指数。
3. 银轮股份、同飞股份等冷板/设备方向有资金承接。
4. CPO/光模块、半导体、存储没有集体补跌。
5. 板块讨论从“液冷概念”转向“冷板、CDU、干冷器、冷却液、快接头”。
6. 中军有成交额，后排不是单纯一字或冲高回落。

### 判断只是短炒

出现以下 3 条以上，按短炒处理：

1. 只有小票一字，中军没量。
2. 液冷高开后 10:00 前快速回落。
3. 冷却液、浸没式后排先涨，冷板/CDU 不跟。
4. AI 硬件主线同时走弱。
5. 板块内部涨停少、炸板多。
6. 大盘缩量，资金抢流动性。

## 九、和当前市场主线的关系

液冷不能脱离 AI 硬件主线单独判断。

当前 A 股科技主线排序仍应是：

```text
指数环境
→ 科技核心锚点
→ 半导体/存储/封测强度
→ CPO/光模块修复
→ Rubin/液冷/PCB/电源等扩散
```

也就是说：

- 如果中际旭创、新易盛、天孚通信继续修复，液冷更容易被当作 AI 硬件扩散。
- 如果长电科技、佰维存储、江波龙等业绩线继续强，液冷属于科技内部轮动。
- 如果科技核心集体补跌，液冷后排即使高开，也容易冲高回落。

## 十、最终结论

英伟达这次高温液冷新技术，把 Rubin 液冷逻辑从“是否需要液冷”推进到“需要什么样的液冷系统”：

**高温 Direct-to-Chip 冷板液冷 + CDU + 干冷器 + 防泄漏冷却液 + 管路快接密封。**

A 股炒作顺序应按：

```text
英维克 / 申菱环境等中军确认
→ 银轮股份 / 飞荣达 / 金富科技等冷板液冷
→ 高澜股份 / 同飞股份等 CDU 与设备弹性
```
