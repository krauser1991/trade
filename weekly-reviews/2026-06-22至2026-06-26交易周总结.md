# 2026-06-22 至 2026-06-26 交易周总结

生成时间：2026-06-27  
数据来源：本周资金曲线、每日持仓快照、盘后复盘文档。  
用途：复盘本周交易行为，提炼可复用纪律和下周改进方向。  
结论属性：个人交易复盘，不构成投资建议。

## 一、本周一句话结论

本周最重要的进步不是赚了多少钱，而是：**从高仓位、弱票拖累、T+1 被动，逐步切换到低仓位、留强去弱、周五主动空仓避险。**

账户从 2026-06-20 的 512,740.22 提升到 2026-06-26 的 532,819.72，增加 20,079.50，阶段收益约 +3.92%。如果只看周一收盘到周五收盘，账户从 530,089.14 到 532,819.72，增加 2,730.58，说明本周主要收益来自周一强修复，后四天核心任务其实是 **控制回撤和纠错**。

本周最大亮点：

```text
周五大跌日主动降到 1.3% 仓位
账户在创业板 -4.07% 当天仍盈利 +0.67%
```

本周最大问题：

```text
买题材逻辑多，买资金最认可的核心少
强市里持仓跑不赢，弱市里容易想低吸
```

## 三、本周市场环境

本周不是单边主升，而是典型的高波动科技周：

```text
周一：科技强修复
周二：强修复后分歧杀跌
周三：科技修复，半导体/封测回血
周四：科技强修复但内部分化
周五：指数系统性大跌，科技高位释放风险
```

主线变化：

| 方向 | 本周表现 | 交易结论 |
|---|---|---|
| 封测/先进封装 | 长电科技连续强势，是账户核心利润来源 | 可以做强锚，但不能涨停后追高加 |
| 半导体材料 | 沪硅产业从弱试错变成周五逆势观察仓 | 可观察，但必须看强弱确认 |
| CPO/光模块 | 高位分歧明显，中际、新易盛、天孚周五大跌 | 不适合后手接高位 |
| PCB/服务器链 | 周四强，周五大跌，胜宏/沪电/生益释放风险 | 需要等止跌确认 |
| AI电感/被动元件 | 麦捷弱于顺络，逻辑不等于资金认可 | 不能再硬讲逻辑 |
| 光纤光棒/空芯光纤 | 大族公告刺激后冲高回落，亨通/中天重挫 | 不能接公告兑现和板块杀跌 |
| 液冷 | 英维克等有阶段表现，但本周不是阻力最小方向 | 只能尾盘小仓试错，不做主仓 |

## 四、每日复盘

### 2026-06-22：强修复日，赚钱后降仓

账户结果：

- 总资产 530,089.14；
- 当日盈利 +17,559.84；
- 仓位从此前高位降到 39.9%。

做对的地方：

- 长电科技、利通电子等强势利润仓保留到位。
- 在账户大赚时执行降仓，没有因为市场强就重新打满。
- 弱票有清理动作，账户从高仓位状态转向更主动。

问题：

- 新仓数量仍偏多，方向扩得有点散。
- 利通、长电这种大浮盈仓需要更明确的保护规则。

本日评价：

**赚钱质量较高，最重要的动作是降仓。**

### 2026-06-23：分歧日，T+1 弱仓暴露

账户结果：

- 总资产 524,977.06；
- 当日亏损 -4,958.12；
- 仓位降到 30.0%。

主要问题：

- 飞荣达、双杰电气、沪硅产业等新仓或加仓后可用为 0，盘中无法纠错。

做对的地方：

- 总仓位已降，亏损没有扩大成账户级伤害。
- 开始意识到“逻辑正宗但分时弱，就是弱”。

本日评价：

**这是本周最典型的教训日：分歧日不该铺太多 T+1 新仓。**

## 五、本周做对的三件事

### 1. 仓位一路下降，风险逐步收缩

本周仓位变化：

```text
71.9% → 39.9% → 30.0% → 24.9% → 16.3% → 1.3%
```

这条线非常重要。它说明你没有在高波动科技周里越亏越补，也没有在强修复日越涨越上头。

但这里要补一句更准确的话：**仓位低不是目标，仓位和市场状态匹配才是目标。**

如果市场进入主升、核心票持续确认、量能能够支撑扩散，仓位过低会直接降低收益。本周仓位下降是阶段性正确，因为本周后半段科技高位分歧、周五系统性大跌、周末消息不确定；但这不代表以后仓位越低越好。

真正要追求的是：

```text
风险释放期低仓位
弱修复期小仓位
主线确认期中仓位
主升共振期敢于加仓
退潮期果断降仓
```

也就是说，本周做对的是“该降的时候降”，不是“永远低仓位”。

### 2. 对弱票的处理比以前果断

飞荣达、双杰、昭衍、麦捷、大族等票，最终都没有被拖成信仰仓。

尤其是大族激光：

```text
公告刺激
→ 盘中冲高
→ 分时回落
→ 收盘兑现
```

你没有因为“空芯光纤/北美敞口/行业景气”这些逻辑去硬拿，这是很大的进步。

### 3. 周五完成了主动空仓

很多时候账户真正拉开差距，不是靠每天都买对，而是在该躲的时候躲掉。

周五如果你继续拿麦捷、大族、长电，账户大概率会被指数和高位科技拖下去。最终只留沪硅 200 股，这是本周最好的风险动作。

## 六、本周主要问题

### 1. 强市跑输，说明选股强度不足

6 月 25 日是最好的镜子：

```text
科创50 +3.87%
创业板 +2.84%
账户 -0.12%
```

这说明问题不是市场不给机会，而是你手里的票不是资金最认可的核心。

后续每次买票前要问：

```text
如果这个方向真的强，资金为什么要买我这只？
它是容量锚、情绪锚、逻辑锚，还是只是后排映射？
同题材里有没有更强的票？
```

### 2. T+1 试错仓仍然偏多

6 月 23 日的问题很典型：分歧日买入多个不可卖新仓，第二天被迫处理。

以后分歧日的试错规则要改成：

- 新仓不超过 1-2 只；
- 单票仓位不超过 3%-5%；
- 必须买当日主动强，不买弱修复；
- 如果不是尾盘确认，不轻易隔夜。

### 3. 容易用逻辑替代盘面

本周反复出现：

- 麦捷有 AI 电感逻辑，但弱于顺络；
- 大族有空芯光纤公告，但弱于德龙和芯碁；
- 沪硅有半导体材料逻辑，但前半周弱于科创和封测核心。

交易上要记住：

```text
逻辑只是入场理由
强度才是持仓理由
承接才是加仓理由
```

### 4. 空仓后不够舒服

周五盘中空仓后，仍然不断寻找低吸票。这个问题很真实。

空仓不是踏空，空仓也是一种仓位。特别是在：

- 周五；
- 大跌日；
- 下周一有银行出金日预期；
- 周末消息不确定；
- 科技高位核心未止跌；

这种环境里，空仓就是主动权。

### 5. 仓位过低也会错过收益

本周也出现了另一个反面提醒：6 月 25 日科技强修复，长电科技涨停，兆易创新接近涨停，新易盛接近涨停，沪电股份、生益科技大涨，但账户没有跟上。

原因不只是选股问题，也有仓位结构问题：

- 强票长电科技仓位只有约 2.0%；
- 低仓位虽然降低了风险，但也降低了强修复日的收益弹性；
- 后排弱票占用了注意力，但真正强锚仓位不够；
- 如果市场已经确认修复，仍然过度防守，就会变成“看对方向但赚不到钱”。

所以仓位不是越小越好。下周要解决的是：

```text
弱市敢空
强市敢上
分歧少试
确认加仓
```

低仓位是防守工具，不是交易模式本身。

## 七、本周核心个股复盘

### 长电科技

本周定位：封测核心利润仓。

做对：

- 周一、周三吃到强势；
- 大部分时间没有把它当弱票随便处理；
- 周五最终兑现，保护了利润。

不足：

- 强的时候仓位偏小，导致强修复日贡献有限。
- 涨停后没有形成清晰的“底仓/交易仓”拆分计划。

后续规则：

**长电这类核心强锚，可以留底仓，但不能涨停后追高加。**

### 沪硅产业

本周定位：半导体材料试错到观察仓。

做对：

- 前半周没有无限补弱；
- 周五保留 200 股小仓观察，既有盘感又风险很小。

不足：

- 前期买入时强度不够，曾弱于科创50和半导体核心。

后续规则：

**沪硅只有在强于科创50、强于材料链时才值得提高仓位；否则只当观察仓。**

### 麦捷科技

本周定位：AI电感/服务器BOM试错失败。

问题：

- 逻辑有，但同题材顺络更强。
- 强科技日反而弱，说明资金不认可它是核心。

后续规则：

**麦捷不能按长鑫、AI服务器泛逻辑硬拿，只能看顺络和板块承接。**

### 大族激光

本周定位：公告刺激/空芯光纤跨界逻辑票。

问题：

- 盘后公告确实增加想象力；
- 但周五高开冲高后大幅回落，说明公告被兑现；
- 它不是北美 AI 光纤核心票，也不是玻璃桥设备最强弹性票。

后续规则：

**大族只有重新站回强势区并获得资金确认，才有交易价值；公告不能替代盘面。**

### 飞荣达

本周定位：液冷试错失败。

问题：

- 英维克、高澜强时，飞荣达不强；
- 同题材里没有买到阻力最小方向。

后续规则：

**同题材中军强、自己弱，就是弱；不能靠题材正宗硬拿。**

## 八、收益率为什么还是上不去

本周其实已经给出答案：

### 1. 核心票仓位小，后排票数量多

长电科技强，但仓位不大；麦捷、大族、沪硅这些观察/试错仓占了不少注意力。

结果就是：

```text
强票涨，对账户贡献不够
弱票跌，对心态拖累很大
```

### 2. 试错太分散

一周里出现过：

- 长电；
- 利通；
- 沪硅；
- 飞荣达；
- 双杰；
- 昭衍；
- 世纪恒通；
- 行云科技；
- 麦捷；
- 大族。

方向太多，交易重心就会散。你会一直在处理票，而不是集中做最强方向。

### 3. 对“强”的要求还不够高

真正值得上仓位的票，需要满足：

- 板块强；
- 自己强；
- 同题材里排前；
- 有成交承接；
- 次日还能验证；
- 不是单纯消息刺激。

本周很多票只满足“有逻辑”，但不满足“强度排序靠前”。

## 九、下周交易纪律

### 1. 仓位纪律

下周默认从低仓位开始。

| 市场状态 | 仓位 |
|---|---:|
| 周一继续弱 | 0%-5% |
| 弱修复 | 0%-10% |
| 指数修复但核心不强 | 10%-20% |
| 科技核心和指数共振修复 | 20%-30% |
| 放量主升确认 | 再考虑 30% 以上 |

当前不是必须急着赚钱的阶段。周五收盘你只剩 1.3% 仓位，主动权很好，不要轻易送回去。

更完整的仓位原则：

| 阶段 | 仓位目标 | 核心动作 |
|---|---:|---|
| 退潮/杀跌 | 0%-10% | 保护本金，等待止跌 |
| 弱修复 | 10%-20% | 只做最强 1-2 只试错 |
| 强修复但未确认主升 | 20%-40% | 核心票确认后逐步加，不铺后排 |
| 主线确认且量能支持 | 40%-60% | 把仓位给核心，不分散乱买 |
| 明确主升/账户顺风 | 60% 以上 | 只在模式内加仓，且必须有止盈止损计划 |

低仓位可以保护你，但无法让账户快速增长。真正的目标是：**在不确定时小，在确定性提高时敢于把仓位给最强核心。**

### 2. 买入纪律

下周新开仓必须满足至少 3 个条件：

- 是当天阻力最小方向；
- 板块有容量锚或情绪锚；
- 个股强于同题材多数票；
- 不是高开冲高回落；
- 买点在分歧承接或尾盘确认，不是盘中冲动；
- 买入后次日有明确处理线。

### 3. 试错纪律

试错仓规则：

```text
单票 3%-5%
总试错仓不超过 10%
一天最多试 1-2 只
分歧日尽量尾盘买
买错次日直接处理
不补弱，不摊平
```

### 4. 持仓纪律

持仓分三类：

| 类型 | 处理 |
|---|---|
| 核心强锚 | 可留底仓，分歧看承接 |
| 试错仓 | 不主动转强就走 |
| 后排逻辑票 | 只做短线，不讲信仰 |

### 5. 空仓纪律

下周继续记住：

**空仓不是没操作，空仓是在等更好的阻力最小点。**

尤其在以下情况，允许空仓：

- 周末消息不确定；
- 周一银行出金日影响仍在；
- CPO/PCB高位未止跌；
- 指数弱修复但核心不强；
- 自己找不到最强票。

## 十一、给自己的周总结

本周你的交易从“有逻辑就想做”，往“盘面不认就处理”迈了一步。

最值得保留的是：

```text
赚钱后降仓
弱票不补
公告不信仰
周五主动空仓
让账户在大跌日仍创新高
```

最需要改的是：

```text
减少试错数量
提高选股强度要求
核心票仓位要匹配强度
分歧日少买 T+1 新仓
空仓后不要急着找票
```

下周最重要的一句话：

**少做一点，做强一点。**

如果市场没有给出清晰修复，就继续保持低仓位；如果市场给出修复，也只买最强方向里的最强票，不再用后排逻辑票消耗账户和注意力。
