# 俞亮亮的交易体系 0628 版

生成日期：2026-06-28  
适用账户：A 股个人账户，权益资金约 50 万  
文档性质：个人交易体系、复盘纪律和执行清单，不构成投资建议。

## 一、总纲

我的交易体系只有一句话：

**先判断大盘状态，再推演资金路径，只做阻力最小的主线核心；错了快速纠错，对了让强票承担主要收益。**

这套体系不是为了每天交易，也不是为了买到所有热点，而是为了让账户长期保持三种能力：

- 市场不确定时，能空仓和低仓等待；
- 市场确认时，敢把仓位给最强核心；
- 判断错误时，能小亏退出，不把短线失败拖成长线信仰。

我真正要追求的不是“每次都买对”，而是：

```text
看对市场状态
买在资金认可方向
仓位匹配确定性
止损提前定义
复盘持续纠偏
```

## 三、大盘状态系统

交易第一步不是问买什么，而是问市场处在哪个状态。

### 1. 四个温度

每天盘前和盘中都要看四个温度：

| 温度 | 观察项 | 含义 |
|---|---|---|
| 指数温度 | 上证、创业板、科创50、主要均线 | 决定风险偏好 |
| 量能温度 | 成交额、日内量能曲线、午后是否缩量 | 决定能否多线进攻 |
| 广度温度 | 上涨家数、涨跌停、炸板率 | 判断普涨、分化还是退潮 |
| 题材温度 | 强势股数量、核心锚点、前排承接 | 判断赚钱效应是否还在 |

### 2. 四种市场状态与仓位

| 状态 | 特征 | 仓位上限 | 动作 |
|---|---|---:|---|
| 退潮/缩量抢流动性 | 指数弱、炸板多、核心锚转弱、午后缩量 | 0%-20% | 空仓或观察仓 |
| 弱指数抱团/局部轮动 | 指数弱但强势股仍多，钱集中少数方向 | 20%-40% | 只做前排和容量锚 |
| 放量修复/低位反弹 | 成交改善，核心分歧后能修复 | 40%-65% | 主攻阻力最小方向 |
| 主升确认 | 指数和题材共振，容量锚持续走强 | 65%-85% | 集中主线核心 |

0628 版特别修正：

**仓位低不是目标，仓位匹配市场状态才是目标。**

本周做对的是风险释放时降仓，不是永远低仓。以后要做到：

```text
弱市敢空
强市敢上
分歧少试
确认加仓
退潮快退
```

## 四、资金路径系统

拆哥模式的核心是：**找阻力最小的地方。**

阻力最小不是逻辑最动听，而是资金最容易形成共识、持筹者卖压较小、市场最容易推得动。

### 1. 主线、承接、轮动

| 类型 | 特征 | 交易含义 |
|---|---|---|
| 主线 | 有容量锚、有持续赚钱效应、分歧后能修复 | 可以持仓、做 T、适度加仓 |
| 承接 | 原主线分歧后，资金暂时找的新地方 | 小仓试错，等分歧验证 |
| 轮动 | 板块短期冲高，持续性未确认 | 不重仓，不追后排 |

一个承接方向要升级为主线，必须经过分歧日验证：

- 分歧日不跟跌；
- 有容量票放量；
- 前排强于后排；
- 同一细分链条集体走强；
- 次日仍有溢价和承接。

### 2. 资金迁移顺序

常见迁移路径：

```text
赚钱效应出现
→ 容量锚确认
→ 产业链扩散
→ 高位钝化
→ 高低切
→ 新解释出现
→ 再扩散或退潮
```

每天要问：

- 谁替代谁？
- 为什么替代？
- 是真正主线切换，还是旧方向反抽？
- 新方向有没有容量锚？
- 资金是从高位撤出，还是主线内部轮动？

## 五、核心锚点系统

不要看热闹，要看锚点。

| 锚点类型 | 作用 | 示例 |
|---|---|---|
| 指数锚 | 判断风险偏好 | 上证、创业板、科创50 |
| 容量锚 | 判断大资金是否认可 | 中际旭创、工业富联、长电科技 |
| 情绪锚 | 判断短线赚钱效应 | 20cm 核心、一字板核心 |
| 逻辑锚 | 判断产业纯度 | 硅光、封测、材料、设备核心 |
| 板块锚 | 判断细分强弱 | 存储、半导体设备、PCB、液冷核心 |

科技线当前重点锚点：

- 算力/光模块：中际旭创、新易盛、天孚通信；
- 封测：长电科技、通富微电；
- 存储：兆易创新、佰维存储、香农芯创；
- 半导体设备：富创精密、盛美上海、中微公司；
- PCB/CCL：沪电股份、生益科技、胜宏科技；
- 半导体材料：沪硅产业及材料链核心；
- 液冷：英维克、高澜股份、申菱环境。

锚点规则：

- 锚点强，后排才有交易价值；
- 锚点弱，后排先降级；
- 只有后排涨、核心不动，按假承接处理；
- 容量锚放量确认，方向优先级上调；
- 容量锚翻绿，科技仓位先减再看。

## 六、题材拆解与炒作顺序

项目里所有产业链研究最终都指向同一个方法：

**不要买板块名，要拆产业环节和炒作顺序。**

### 1. 已知 BOM 型

适用于 AI 服务器、Rubin、先进封装、长鑫链等已经能拆出价值量的方向。

炒作顺序通常是：

```text
核心容量锚
→ 材料扩散
→ 设备/测试/封装共振
→ BOM 配套补涨
→ 远端分支补充
```

交易原则：

- 先看核心环节是否确认；
- 再看一阶扩散是否跟上；
- 补涨只能轻仓；
- 远端分支只在情绪溢出阶段做。

### 2. 新技术 0 到 1 型

适用于 CPO、硅光、玻璃桥、液冷新方案、特殊材料等。

炒作顺序通常是：

```text
最大增量环节
→ 上游量价环节
→ 工艺瓶颈环节
→ 设备/封装/测试
→ 重叠受益方向
→ 情绪补涨
```

交易原则：

- 首日不盲目追远端；
- 优先找最直接受益和最能解释增量的标的；
- 后排必须看前排强度；
- 没有订单、认证、客户、产能、价格事实，不能按主线重仓。

### 3. IPO/新故事承接型

适用于 MiniMax、智谱、软件类 IPO 等。

炒作顺序不是硬套供应链，而是：

```text
高位硬件分歧
→ 资金寻找低位科技承接
→ IPO/政策/会议提供新解释
→ 分歧日验证独立性
→ 容量票确认后才升级
```

交易原则：

- 不把新故事直接当新主线；
- 等一次分歧，看能否抗跌；
- 没有容量票，不能重仓；
- 软件型 IPO 不硬套制造型产业链。

## 七、当前科技主线 0628 版理解

科技行情不是单一 CPO 主线，而是三大业绩要素轮动：

```text
算力
存储
半导体设备
```

短期判断：

- 科技线继续强，需要指数支持；
- 美联储决议等外围变量落地后，可能出现利好兑现；
- 节前仓位不宜过满，四成左右更适合对抗波动；
- 若高开低走，防利好兑现；
- 若承接后再次走强，保留核心仓位；
- 如果只有单点强、板块不扩散，降低进攻仓位。

当前选股偏好：

- 新高附近；
- 突破形态；
- 连阳形态；
- 分歧后率先修复；
- 强于同题材多数票；
- 有容量锚或板块灵魂地位。

弱图形降级：

- 箱体震荡；
- 弱于中军；
- 公告刺激冲高回落；
- 高开低走无承接；
- 逻辑正宗但资金不选。

## 八、仓位系统

### 1. 账户分层

以 50 万账户为基础：

| 资金层 | 金额 | 用途 |
|---|---:|---|
| 防守现金 | 10 万 | 防止情绪失控和极端机会 |
| 主作战资金 | 30 万 | 主线确认后的核心仓位 |
| 试错资金 | 10 万 | 新方向、低位承接、事件验证 |

### 2. 单票仓位

| 类型 | 金额 | 说明 |
|---|---:|---|
| 试错仓 | 1 万-2.5 万 | 错了不伤账户 |
| 普通确认仓 | 3 万-5 万 | 方向确认但非最高确定性 |
| 高确定性核心仓 | 5 万-7.5 万 | 主线核心、容量锚 |
| 单票极限 | 10 万 | 只给最熟悉、最核心方向 |

### 3. 试错规则

```text
单票 3%-5%
总试错仓不超过 10%
一天最多试 1-2 只
分歧日尽量尾盘买
买错次日直接处理
不补弱，不摊平
```

### 4. 仓位和心态

会议纪要补充的重要规则：

- 七成以上仓位容易恐慌卖飞；
- 五成以下更利于持股；
- 节前不确定时四成左右更舒服；
- 两成以下容易踏空焦虑，只适合风险显著放大阶段。

所以仓位不是越低越好，也不是越高越勇敢。

**仓位要让自己既能执行计划，又能分享行情。**

## 九、买入系统

买入前必须回答七个问题：

1. 当前大盘状态是什么？
2. 当前钱够不够？
3. 这是主线、承接，还是轮动？
4. 容量锚是谁，强不强？
5. 我买的是核心、前排、一阶扩散，还是后排？
6. 如果错了，跌到哪里走？
7. 这笔最多亏多少钱？

### 允许买入的三类机会

| 类型 | 条件 |
|---|---|
| 主线确认后的核心 | 大盘支持、容量锚强、分歧后有承接 |
| 旧主线松动后的低位承接 | 原主线高位钝化，新方向有辨识度和容量 |
| 低位反弹阶段的阻力最小方向 | 位置低，有业绩、涨价、产业催化，资金反复选择 |

### 禁止买入

- 大盘缩量退潮；
- 午后资金抢流动性；
- 核心锚点翻绿，后排却乱涨；
- 只因为看到涨停；
- 只因为群里讨论热；
- 只因为不买怕错过；
- 没有止损点；
- 当天亏损失控还想扳回；
- 分歧日同时买多个 T+1 新仓；
- 公告刺激高开冲高回落还硬接。

## 十、卖出与止损系统

### 1. 止损底层逻辑

买入是基于某个标准：

- 逻辑；
- 预期；
- 图形；
- 资金强度；
- 事件节点；
- 板块地位。

当标准被破坏，就要卖。

止损不是亏钱后的临时反应，而是买入计划的一部分。

### 2. 技术生命线

| 类型 | 生命线 | 处理 |
|---|---|---|
| 强势股 | 5 日线 | 有效跌破且无反包迹象，减仓或离场 |
| 趋势股 | 10 日线 | 跌破需警惕调整周期延长 |
| 核心趋势仓 | 20/30 日线 | 跌破 20 日线减交易仓，跌破 30 日线降趋势仓 |
| 长趋势失效 | 60 日线 | 跌破且逻辑走弱，退出趋势仓 |

### 3. 防守卖出

出现以下任一情况，先卖或减：

- 买入逻辑证伪；
- 跌破预设止损；
- 核心锚点转弱；
- 主线前排炸板且无法修复；
- 大盘进入缩量抢流动性；
- 后排冲高无承接；
- 发现自己无法解释这笔交易；
- 短线交易失败却想改成长线。

### 4. 盈利卖出

- 轮动票冲高不强就卖；
- 承接票第一次分歧不抗跌就卖；
- 主线核心只要锚点不破、量能不坏，可以持有或做 T；
- 单票盈利超过 20%，至少把本金风险降下来；
- 盈利来自消息刺激，不把它当能力。

## 十一、加仓系统

加仓只能发生在正确之后，不能发生在亏损之后。

允许加仓：

- 初始买入盈利；
- 大盘状态继续改善；
- 主线容量锚继续强；
- 分歧后承接好；
- 加仓后总风险仍可控。

禁止加仓：

- 浮亏摊成本；
- 因为不甘心；
- 因为跌多了觉得便宜；
- 核心锚点转弱；
- 大盘缩量；
- 试错仓失败后补仓。

加仓节奏：

```text
第一笔试错：20%-30%计划仓位
确认后第二笔：30%-40%计划仓位
走出利润后第三笔：剩余仓位
加仓后抬高止损，不允许利润变亏损
```

## 十二、心态系统

### 1. 卖飞

卖飞不等于失败。

能红盘卖出已经超过多数投资者。真正的错误不是卖飞，而是：

- 为了追求最高点，把盈利拖成亏损；
- 卖飞后报复性买回；
- 因为别人赚钱，打乱自己的系统；
- 在没有买点时追高补票。

### 2. 踏空

踏空感来自幸存者偏差。

群里看到的是别人赚钱的片段，不是所有人的真实收益分布。

应对方法：

- 少看炫耀信息；
- 回到月度稳定盈利目标；
- 用复盘证明自己是否执行正确；
- 不把别人赚钱当成自己的亏损；
- 空仓时明确自己在等什么。

### 3. 恐高

恐高不是靠喊口号解决，而靠低成本训练。

可以从小仓位开始：

```text
2000 元
→ 5000 元
→ 1 万
→ 5 万
```

只要是模式内强势股，就用小仓位训练对高位波动的承受力。

### 4. 收益率上不去的真正原因

本周复盘给出答案：

- 核心票仓位小，后排票数量多；
- 试错太分散；
- 对“强”的要求还不够高；
- 逻辑替代盘面；
- 分歧日买太多 T+1 新仓；
- 空仓后不舒服，急着找票。

以后解决方式：

```text
少做一点
做强一点
强票给仓位
弱票快处理
试错更机械
空仓更舒服
```

## 十三、T+1 与盘中执行系统

A 股 T+1 最大的问题是：当天买错，盘中不能纠错。

所以：

- 分歧日不铺新仓；
- 新仓不超过 1-2 只；
- 单票 3%-5%；
- 尽量尾盘确认再买；
- 如果不是强承接，不轻易隔夜；
- 买错次日直接处理。

盘中观察顺序：

| 时间 | 观察重点 |
|---|---|
| 9:15 | 封单参考价值低，量化虚假挂单多 |
| 9:20 | 撤单后封单，初步判断真实强度 |
| 9:24:50 | 临近开盘封单，更接近真实资金态度 |
| 9:30-10:00 | 看承接，不急着买 |
| 10:00 后 | 判断胜出主线 |
| 午后 | 看量能是否继续 |
| 14:00 后 | 警惕抢流动性 |

突发题材强度：

- 三个以上一字板；
- 包含 20cm 一字板；
- 有容量锚确认；
- 同细分链条集体走强；
- 不是只有后排小票乱拉。

## 十四、个股案例沉淀

### 长电科技

封测核心利润仓。可以作为强锚留底仓，但不能涨停后追高加。后续要提前拆分底仓和交易仓，避免强的时候仓位太小、弱的时候又没有处理计划。

### 沪硅产业

半导体材料观察仓。只有强于科创50、强于材料链时才值得提高仓位；否则只是观察仓，不能因为逻辑正宗就升级为核心。

### 麦捷科技

AI 电感/服务器 BOM 试错失败。顺络强而麦捷弱，说明资金没有选择它。后续不能按泛 AI 服务器逻辑硬拿，只看同题材强弱排序和资金承接。

### 大族激光

公告刺激和空芯光纤跨界逻辑票。公告不能替代盘面，冲高回落就是资金兑现。只有重新站回强势区并获得资金确认，才有交易价值。

### 飞荣达

液冷试错失败。同题材英维克、高澜强，飞荣达弱，说明没有买到阻力最小票。以后同题材中军强、自己弱，直接按弱处理。

### 华工科技、大族激光价值报告启示

好公司不等于好价格。价值投资要看安全边际，短线交易要看资金承接。公司质地、产业逻辑、盘面强度、买点价格，是四件不同的事。

## 十五、每日执行清单

### 盘前

- 今天大盘状态是什么？
- 今天最多允许几成仓？
- 昨日主线是加强、兑现，还是分歧？
- 今天阻力最小方向是什么？
- 今天核心锚点是谁？
- 哪些方向只观察不买？
- 哪些票跌破什么位置必须处理？

### 盘中

- 9:30-10:00 只观察承接；
- 10:00 后确认胜出方向；
- 看成交量是否急缩；
- 看核心锚点是否翻绿；
- 看前排强于后排还是后排乱拉；
- 午后看是否抢流动性；
- 手痒离开交易界面 10 分钟；
- 当天亏损到纪律线，停止交易。

### 收盘

- 今天大盘判断对了吗？
- 今天仓位和市场状态匹配吗？
- 今天买的是核心还是后排？
- 今天有没有分歧日买太多 T+1 新仓？
- 今天有没有用逻辑替代盘面？
- 今天有没有把试错仓拖成信仰仓？
- 今天空仓是否焦虑？
- 今天强票仓位是否匹配强度？

## 十六、0628 版红线纪律

违反一次，停止交易一天：

- 随手单；
- 不设止损买入；
- 浮亏加仓；
- 把短线套牢改成长线；
- 大盘退潮还重仓进攻；
- 分歧日乱买多个 T+1 新仓；
- 买入自己无法解释的票；
- 亏损后报复性交易；
- 后排逻辑票重仓；
- 公告冲高回落还讲故事。

## 十七、0628 版最终结论

我不是靠预测吃饭，而是靠系统吃饭。

我的系统不是为了抓住所有机会，而是为了：

- 过滤掉大部分无效机会；
- 在真正强的方向出现时敢做；
- 在市场不支持时敢空；
- 在错的时候亏小钱；
- 在对的时候让核心仓位贡献主要收益。

0628 版最重要的一句话：

**少做一点，做强一点；弱市敢空，强市敢上；逻辑只是入场理由，强度才是持仓理由，承接才是加仓理由。**
