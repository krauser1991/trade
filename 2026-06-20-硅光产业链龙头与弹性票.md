# 2026-06-20 硅光产业链龙头与弹性票

> 用途：给同花顺自选分组、盘前主线判断和盘后复盘使用。  
> 结论先行：A股目前没有很多“纯硅光芯片”公司，二级市场主要炒作的是`硅光子 + CPO + 高速光模块 + 耦合测试设备 + 半导体制程设备`的组合链条。

## 一、硅光的炒作价值

硅光不是普通光模块的一个分支，而是 AI 算力基础设施继续扩张后，对“带宽、功耗、密度、成本”四个瓶颈的系统性解法。因此它的炒作价值来自三层叠加：`AI算力需求外溢 -> 光互联升级 -> 硅光/CPO产业化预期`。

### 1. AI数据中心的带宽瓶颈给了硅光大叙事

AI集群越大，GPU之间、机柜之间、数据中心内部的互联压力越高。传统电互联在高速、长距离、低功耗上的压力越来越明显，光互联从“通信配套”变成“算力系统瓶颈”。这就是硅光最核心的叙事：它不是单独卖一个器件，而是绑定 AI 数据中心下一代架构升级。

盘面含义：

- 只要 AI 算力主线还在，市场就会反复寻找光通信里的下一代技术方向。
- 800G、1.6T、CPO、硅光、光I/O会形成连续炒作链。
- 中际旭创、新易盛、天孚通信越强，硅光扩散的土壤越好。

### 2. CPO把硅光从“远期技术”变成“可交易预期”

CPO的核心是把光模块/光引擎更靠近交换芯片、计算芯片，降低功耗并提高带宽密度。硅光适合做高集成度、可规模制造的光子芯片平台，因此天然和 CPO 绑定。

盘面含义：

- CPO一旦有海外大厂进展、交换芯片升级、1.6T/3.2T路线讨论，A股会先炒光模块龙头，再扩散到硅光平台、耦合测试设备、半导体制程设备。
- 罗博特科的价值在于“硅光后段自动化装配、耦合、测试”这条链足够具体，不只是蹭概念。
- 光迅科技的价值在于国内光通信平台属性，容易被资金当成硅光国产化代表。

### 3. 硅光有“技术稀缺性 + 国产替代”双标签

A股真正能讲清楚硅光芯片、光芯片、光器件、耦合测试设备的标的不多，稀缺性会放大估值弹性。另一方面，硅光涉及晶圆制造、光电封装、测试设备、材料平台，天然能和国产半导体设备链形成共振。

盘面含义：

- 行情早期：资金先买最确定的高速光模块龙头。
- 行情中期：资金开始挖硅光芯片、光芯片、光器件、设备。
- 行情后期：低位“沾边票”乱飞，风险反而上升。

### 4. 硅光适合形成“龙头定方向，弹性票放大收益”的结构

硅光本身产业链长，但A股纯度参差不齐。更合理的交易方式不是押单只概念股，而是用龙头判断主线强度，用弹性票判断资金扩散。

交易分层：

| 层级 | 代表 | 作用 |
|---|---|---|
| 主线锚 | 中际旭创、新易盛、天孚通信 | 判断AI光通信是否仍是市场主线 |
| 硅光平台 | 光迅科技 | 判断资金是否认可硅光国产平台 |
| 设备核心 | 罗博特科 | 判断资金是否从光模块扩散到硅光/CPO后段设备 |
| 制程确认 | 中微公司、拓荆科技、中科飞测 | 判断是否扩散到半导体设备国产化 |
| 情绪弹性 | 剑桥科技、源杰科技、仕佳光子、光库科技、博创科技 | 判断短线赚钱效应和题材纵深 |

### 5. 最大的炒作价值在“预期差”，不是当期业绩

硅光/CPO很多环节仍处于导入、验证、放量前期，短期业绩未必马上兑现。市场愿意炒，是因为它具备从 800G/1.6T 光模块向下一代 CPO/光I/O升级的路径想象。

要注意两点：

- 如果只是传统光模块涨价或订单景气，硅光弹性票未必跟得上。
- 如果出现CPO产业化、硅光封装、海外AI数据中心光互联升级的强催化，硅光弹性会明显强于普通光通信补涨。

### 6. 硅光题材的风险

- 产业化节奏风险：CPO和硅光大规模部署可能慢于市场预期。
- 纯度风险：很多A股公司只是光通信、设备或检测映射，并非纯硅光。
- 估值风险：光模块核心股涨幅大后，硅光弹性票容易被当成补涨工具，回撤也会更快。
- 订单验证风险：没有客户、订单、良率、产能验证的公司，题材持续性弱。
- 情绪退潮风险：AI光通信主线一旦退潮，硅光作为扩散分支通常跌得更快。

一句话结论：硅光的炒作价值，不在于它今天贡献多少利润，而在于它站在`AI算力扩张后的下一代光互联升级`上；但实战里必须用中际旭创、新易盛、天孚通信确认主线温度，用光迅科技、罗博特科确认硅光扩散强度。

## 二、硅光产业链拆分

### 1. 上游：材料与晶圆制造

关键词：SOI硅片、半导体硅片、薄膜沉积、刻蚀、清洗、CMP、检测量测。

| 环节 | 核心公司 | 说明 |
|---|---|---|
| 半导体硅片/材料 | 沪硅产业、有研硅 | 更偏硅基材料，不是直接硅光芯片公司 |
| 刻蚀设备 | 中微公司 | 硅光波导、光栅等微纳结构离不开刻蚀能力 |
| 薄膜沉积 | 拓荆科技、北方华创 | 硅光平台涉及 Si、SiN、SiO2 等薄膜工艺 |
| 清洗/CMP | 盛美上海、华海清科 | 晶圆制造通用设备，硅光制程映射 |
| 检测量测 | 中科飞测、精测电子 | 良率提升与制程控制映射 |

### 2. 中游：硅光芯片、光芯片、光器件

关键词：硅光芯片、激光器、调制器、AWG/PLC、光无源器件、光引擎。

| 环节 | 核心公司 | 说明 |
|---|---|---|
| 硅光芯片/平台 | 光迅科技 | A股里硅光辨识度较高的产业公司 |
| 激光器芯片 | 源杰科技 | 高速光模块上游光芯片弹性核心 |
| PLC/AWG/无源芯片 | 仕佳光子 | 更偏光无源芯片和器件 |
| 光器件 | 天孚通信、光库科技、太辰光 | 高速光模块/CPO上游器件与连接环节 |

### 3. 后段：封装、耦合、测试与自动化设备

关键词：硅光封装、光电耦合、自动化装配、测试、良率。

| 环节 | 核心公司 | 说明 |
|---|---|---|
| 光子器件自动化装配/耦合/测试 | 罗博特科 | 硅光/CPO设备方向辨识度最高 |
| 自动化检测设备 | 赛腾股份、长川科技 | 偏半导体/电子自动化测试映射 |
| 光学系统/激光光学 | 炬光科技 | 偏光学元件与系统，弹性观察 |

### 4. 下游：CPO、高速光模块、AI数据中心

关键词：800G/1.6T光模块、CPO、AI集群、数据中心互联、海外大客户。

| 环节 | 核心公司 | 说明 |
|---|---|---|
| 高速光模块 | 中际旭创、新易盛 | 最强交易锚，代表AI光模块主线强弱 |
| 光器件平台 | 天孚通信 | 上游器件中军，常与光模块龙头共振 |
| 光通信平台 | 光迅科技、华工科技 | 国内光通信产业链代表 |
| 弹性光模块 | 剑桥科技、博创科技、铭普光磁 | 波动更大，适合板块确认后观察 |

## 六、看盘脚本

1. 先看`中际旭创、新易盛、天孚通信`：三者至少两只强，说明AI光通信主线还在。
2. 再看`罗博特科、光迅科技`：它们决定资金是否从光模块扩散到硅光/CPO设备和硅光平台。
3. 再看`中微公司、拓荆科技、中科飞测`：如果半导体设备也强，说明市场在交易“硅光制造国产化”而不是单纯炒光模块。
4. 弹性票只在龙头确认后看：剑桥科技、博创科技、源杰科技、仕佳光子、光库科技不能先于龙头作为买入依据。
5. 弱市里只保留龙头观察，不扩散到低位杂毛；强市里才看弹性票轮动。
