## 一、核心结论

韬定律V2可以概括为一句话：

```text
只做有主线位置、有业绩验证、有资金锚点、有承接确认的票；
不做只有故事、没有验证、没有锚点、没有赔率的票。
```

在7月中报窗口，韬定律V2的重点不是“题材正宗”，而是“题材能不能被业绩托住”。市场会从上半年的题材扩散，逐步切到：

```text
谁有中报
谁有订单
谁有毛利率改善
谁有Q2环比
谁能高开后承接
谁能带动板块
```

因此，7月选股的优先级应改成：

```text
业绩确定性 > 主线地位 > 锚点强度 > 位置赔率 > 题材纯度
```

这套框架对当前账户尤其重要。经历大回撤后，不能再用高仓位赌远期故事，必须回到“业绩和资金共同确认”的股票。

## 二、韬定律V2的核心公式

韬定律V2的交易评分可以写成：

```text
可交易价值
= 主线级别 x 业绩确定性 x 资金锚点 x 位置赔率 x 承接强度
- 证伪风险
- 拥挤风险
- 账户情绪风险
```

拆开看：

| 因子 | 含义 | 交易解释 |
|---|---|---|
| 主线级别 | 是否处在市场当前最强方向 | 主线票可以给仓位，支线票只能试错 |
| 业绩确定性 | 中报、订单、价格、毛利率是否能验证 | 7月必须把业绩放在题材前面 |
| 资金锚点 | 是否有容量核心带队 | 没有锚点，后排涨停也不可靠 |
| 位置赔率 | 涨幅、回撤、均线、筹码是否合适 | 好逻辑也要有买点 |
| 承接强度 | 高开后是否有买盘，分歧后是否修复 | 交易最终以盘面确认 |
| 证伪风险 | 中报不及预期、现金流差、订单不实 | 一旦证伪，不用解释 |
| 拥挤风险 | 一致预期、高位抱团、监管扰动 | 越一致越要降低追高冲动 |
| 账户情绪风险 | 自己是否急于回本、仓位是否失控 | 账户状态差时，仓位必须降一档 |

## 三、V2相比V1的升级

如果把V1理解为“找到主线，跟随资金”，V2的升级是：

```text
主线只是入场券，
业绩才是7月的安全垫，
锚点决定仓位，
承接决定买点，
证伪决定卖点。
```

### 1. 从题材正宗升级为业绩正宗

过去容易犯的错误是：看到一只票题材正宗，就默认它应该涨。但7月以后，中报会开始验证真假。

更正确的顺序是：

```text
先看主业有没有利润
再看题材有没有增量
再看资金有没有认可
最后看买点是否舒服
```

### 2. 从单票逻辑升级为锚点系统

一只票逻辑再好，如果总锚不稳，也容易冲高回落。

例如：

- 存储要看江波龙、佰维存储；
- 海外链要看中际旭创；
- 超节点要看紫光股份、锐捷网络、盛科通信-U；
- 机器人要看三花智控、拓普集团；
- 电力设备要看金盘科技、思源电气、华明装备。

### 3. 从满仓进攻升级为分层仓位

V2不鼓励“一把梭”。仓位来自确定性，而不是来自情绪。

```text
逻辑有，但盘面未确认：观察或3%-5%
锚点确认，但板块未扩散：5%-10%
锚点确认，板块扩散，中报支撑：10%-15%
阶段主线核心，有利润垫：15%-20%
```

## 四、五个评分维度

韬定律V2建议每只股票给五项评分，每项满分20分，总分100分。

| 维度 | 20分标准 | 低分特征 |
|---|---|---|
| 主线级别 | 属于当下市场最强主线或核心分支 | 只是边缘映射 |
| 业绩质量 | 中报预增、扣非好、毛利率改善、订单确定 | 只有概念，没有利润 |
| 锚点地位 | 是容量锚、情绪锚、产业链核心 | 后排补涨、辨识度低 |
| 位置赔率 | 调整充分或新高突破后承接好 | 高位一致、加速末端 |
| 执行友好 | 有明确买点、止损位和验证条件 | 买入后只能祈祷 |

评分含义：

| 分数 | 定位 | 操作 |
|---:|---|---|
| 85-100 | 主线核心 | 可做核心仓，但仍需分批 |
| 70-84 | 强观察票 | 小中仓试错，确认后加 |
| 55-69 | 轮动票 | 只做低吸或套利 |
| 55以下 | 后排杂毛 | 不做或只观察 |

## 五、中报窗口下的资金选择顺序

7月资金最容易按照下面的顺序选择方向：

```text
第一层：中报已经验证或强预期的业绩线
第二层：前期主线核心回调后的修复
第三层：业绩能扩散的产业链分支
第四层：低位有业绩、有订单、有涨价的补涨
第五层：纯题材、纯小作文、纯后排补涨
```

对应到当前A股：

```text
存储业绩线
-> AI硬件海外链
-> 超节点国产替代
-> 机器人加中报
-> 电力设备出海
-> 后排题材补涨
```

## 七、方向一：存储业绩线

### 1. 为什么最符合韬定律V2

存储是当前最符合V2的方向之一：

```text
主线级别高
+ 中报业绩弹性强
+ 价格和供需有产业支撑
+ 有江波龙、佰维存储等锚点
+ 容易被资金理解
```

项目周末策略中已经把存储排在第一优先级。拆哥会议摘要也强调江波龙业绩超预期，主要来自二季度出货量增加和毛利率提升。

### 2. 核心映射

| 标的 | 逻辑地位 | V2评分 | 交易备注 |
|---|---|---:|---|
| 江波龙 | 存储业绩情绪锚 | 88 | 高开后承接决定板块强度 |
| 佰维存储 | 存储强度锚 | 86 | 看是否与江波龙共振 |
| 香农芯创 | 存储扩散弹性 | 78 | 跟随板块，不能单独重仓 |
| 德明利 | 存储模组弹性 | 76 | 适合强主线里的扩散 |
| 兆易创新 | 存储/MCU容量中军 | 74 | 机构属性更强，弹性弱于模组 |

### 3. 操作条件

存储只有在以下条件成立时才给仓位：

- 江波龙开盘后不快速回落；
- 佰维存储同步承接；
- 香农芯创、德明利有扩散；
- 板块强于机器人和普通半导体；
- 高开后仍有成交量支撑。

## 九、方向三：超节点国产替代

### 1. 核心逻辑

超节点符合V2里的“主线级别”和“锚点系统”，但业绩确定性不如存储，需要盘面验证。

项目内锐捷网络业绩推导已经给出一条路径：

```text
锐捷网络业绩增长
-> 网络设备需求确认
-> 数据中心交换机景气度被关注
-> AI集群和超节点互联逻辑增强
-> 交换芯片/服务器/PCB/连接器/电源液冷扩散
```

### 2. 核心映射

| 标的 | 逻辑地位 | V2评分 | 交易备注 |
|---|---|---:|---|
| 紫光股份 | 国产服务器/超节点容量锚 | 82 | 涨停后看承接，不追一致 |
| 锐捷网络 | 网络设备业绩验证票 | 78 | 有业绩底，但不是总龙头 |

| 中兴通讯 | 网络设备容量中军 | 74 | 弹性较低，胜在容量 |

- 紫光股份不弱；
- 锐捷网络不被兑现；
- 盛科通信-U止跌或走强；
- 寒武纪、海光信息、中芯国际至少有一个企稳；
- 中际旭创不能继续大跌拖累科技风险偏好。

+ 中报不容易证伪
+ 图形强于板块

- A层强于B层时，做稳健锚；
- B层批量联动时，再做弹性；
- 中报不及预期，先卖，不解释。

## 十二、实盘执行规则

### 1. 买入前四问

每次买入前必须回答：

```text
1. 它是不是当前主线或核心分支？
2. 中报或订单有没有验证？
3. 板块锚点是否同步走强？
4. 买点是否有赔率，止损位是否清楚？
```

四个问题缺两个，就不能买。

### 2. 仓位规则

| 类型 | 仓位 |
|---|---:|
| 后排观察票 | 0%-3% |
| 逻辑试错票 | 3%-5% |
| 锚点确认票 | 5%-10% |
| 主线核心票 | 10%-15% |
| 有利润垫的阶段核心 | 15%-20% |

当前账户刚经历过大波动，单票超过15%必须非常谨慎。

### 3. 卖出规则

- 中报不及预期，先卖；
- 高开无承接，红盘减；
- 跌破5日线且无反包，减仓；
- 跌破10日线，按趋势破坏处理；
- 锚点走弱，后排先走；
- 自己开始“找理由硬扛”时，先降仓。

## 十三、最终股票池

### 1. A级：最符合韬定律V2的中报观察核心

| 方向 | 标的 | 关键验证 |
|---|---|---|
| 存储 | 江波龙、佰维存储 | 中报预告、毛利率、出货量、板块承接 |
| 海外链 | 中际旭创、新易盛、天孚通信 | 是否止跌，业绩和订单是否继续强化 |
| PCB | 胜宏科技、沪电股份 | AI服务器订单和利润弹性 |
| 超节点 | 紫光股份、锐捷网络 | 高位承接和网络设备业绩扩散 |
| 机器人业绩锚 | 三花智控、拓普集团 | 主业业绩和机器人增量 |
| 电力设备 | 金盘科技、思源电气 | 出海订单和中报兑现 |

### 2. B级：强观察，等中报或盘面确认

| 方向 | 标的 | 关键验证 |
|---|---|---|
| 存储扩散 | 香农芯创、德明利、兆易创新 | 是否跟随江波龙和佰维 |
| 光纤光缆 | 亨通光电、长飞光纤 | 是否跟随海外链反抽 |
| 超节点扩散 | 盛科通信-U、中兴通讯、浪潮信息、中科曙光 | 是否与紫光、锐捷共振 |
| 机器人弹性 | 五洲新春、北特科技、贝斯特、绿的谐波、鸣志电器 | 中报和订单验证 |
| 电源液冷 | 麦格米特、英维克 | AI服务器功耗扩散是否成立 |

### 3. C级：只做短线扩散，不做主仓

| 方向 | 标的 | 原因 |
|---|---|---|
| 机器人后排 | 奥比中光、兆威机电、柯力传感、拓斯达 | 题材弹性有，但业绩验证要求更高 |
| 后排服务器BOM | 低位补涨票 | 必须看核心锚，不可单独幻想 |
| 纯小作文题材 | 无公告、无业绩、无承接标的 | 中报窗口容易被证伪 |

最终结论：

```text
韬定律V2不是让你买更多票，
而是让你少买弱票，
只在主线、业绩、锚点、位置、承接同时过关时出手。

7月最优先方向：
存储业绩线 > AI硬件海外链 > 超节点国产替代 > 机器人加业绩 > 电力设备出海。

最重要的纪律：
中报窗口里，题材只是加分项，业绩和承接才是决定项。
```
