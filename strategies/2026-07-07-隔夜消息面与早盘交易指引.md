## 一、早盘一句话结论

今天早盘不是进攻日，先按账户修复日处理。

核心策略：

```text
先把亨通光电和总仓位降下来，
再观察存储、海外链、超节点谁被资金选择，
不在锚点未确认前继续满仓赌修复。
```

当前账户最大问题不是没有题材，而是：

- 仓位 99.5%，几乎没有机动资金；
- 亨通光电单票约 38.1%，且浮亏约 9.03%；
- 盈利票有利润垫，但不能对冲最大弱仓；
- 昨天刚写完镜子报告，今天必须验证执行力。

所以今天早盘第一目标是：

```text
把账户从被动满仓，恢复到 60%-70% 以内的可操作状态。
```

如果大盘和核心锚都弱，仓位目标进一步降到 50%-60%。

## 二、隔夜消息面分层

### 1. 第一层：中报业绩线仍是最有确定性的资金方向

项目内本周题材评分显示，当前最容易被资金聚焦的是存储，评分 88 分，原因是：

- 业绩确定性强；
- Q2 出货和毛利改善容易被市场理解；
- 江波龙、佰维存储是清晰锚点；
- 可向香农芯创、德明利、兆易创新、长鑫链、封测扩散。

早盘判断标准：

| 锚点 | 判断意义 |
|---|---|
| 江波龙 | 存储业绩线是否成立 |
| 佰维存储 | 存储板块是否共振 |
| 香农芯创、德明利 | 资金是否从龙头扩散 |
| 兆易创新 | 机构容量资金是否认可 |
| 长电科技、通富微电 | 封测后周期是否跟随 |

结论：存储是今日第一观察方向，但不能竞价追。必须看 9:30 后承接。

### 2. 第二层：海外链有反抽预期，但必须看中际旭创

拆哥分享里，海外链的关键词是“调整后轮动反抽”。其中光模块、PCB、光纤光缆、电子布、钻针都有修复可能。

但海外链能不能做，不由亨通光电单独决定，而由中际旭创决定。

早盘判断标准：

| 锚点 | 判断意义 |
|---|---|
| 中际旭创 | 海外链总锚，不企稳就不重仓 |
| 新易盛、天孚通信 | 光模块扩散质量 |
| 胜宏科技、沪电股份、生益科技 | PCB/AI服务器材料强度 |
| 亨通光电、长飞光纤 | 光纤光缆是否跟随 |
| 麦格米特、英维克 | 电源液冷扩散是否轮动 |

结论：海外链今天可以观察反抽，但你手里的亨通光电不能因为“方向有逻辑”继续做第一大仓。

### 3. 第三层：超节点交换机仍有博弈价值，但看承接

超节点的核心不是再讲故事，而是看紫光股份、锐捷网络、盛科通信-U能否继续承接。

早盘判断标准：

| 锚点 | 判断意义 |
|---|---|
| 紫光股份 | 国产替代容量锚 |
| 锐捷网络 | 交换机视觉锚 |
| 盛科通信-U | 交换芯片弹性锚 |
| 海光信息、寒武纪、中芯国际 | 科技风险偏好锚 |
| 中兴通讯、浪潮信息、中科曙光 | 容量资金是否扩散 |

结论：超节点今天不是无脑追，只有紫光、锐捷、盛科至少两个强，才算方向有效。

### 4. 第四层：机器人和医药只作轮动观察

机器人本周评分 71，医药评分 66，属于轮动观察，不是今天早盘主攻。

原因：

- 机器人高水位，连续性不如业绩线；
- 需要叠加中报业绩和硬件主链，不能追情绪；
- 医药更偏防御和事件轮动；
- 如果科技线修复，资金优先看存储、海外链、超节点。

结论：今天不要用机器人或医药来逃避旧仓位问题。

## 五、持仓处理方案

### 1. 亨通光电：第一处理对象

定位：海外链/光纤光缆方向，但当前是账户最大弱仓。

早盘处理：

| 盘面情况 | 操作 |
|---|---|
| 高开到 86 附近但量能一般 | 先减仓，至少降一半 |
| 放量站上 86.5-88 | 仍然减仓，只保留观察仓 |
| 低开低走跌破 83.5 | 果断降仓，不补仓 |
| 中际旭创强修复，长飞/中天/亨通共振 | 可留小仓观察，但不能继续第一大仓 |

执行目标：

```text
亨通光电从 38.1% 降到 15%-20% 以下。
如果早盘弱，降到 10% 以下。
```

### 2. 甬矽电子：利润垫持仓，不能让盈利变亏

甬矽电子属于先进封装/封测扩散，当前有利润垫。

早盘处理：

- 长电科技、通富微电、江波龙、佰维存储强，则可继续持有；
- 如果高开冲高回落，先保护一部分利润；
- 如果跌回成本附近，不允许盈利票变亏损。

### 3. 汇金科技：题材弹性仓，冲高无力要兑现

汇金科技不属于本周最强硬科技业绩线，更偏题材弹性。

早盘处理：

- 强势高开且继续放量，可观察；
- 冲高无力，先兑现部分；
- 跌破 17.6 附近，降低仓位。

### 4. 利通电子：趋势仓，保护利润

利通电子有利润垫，但不能在满仓状态下无条件持有。

早盘处理：

- 沿 5 日线继续强，可持有；
- 高开回落，保护利润；
- 跌破 170 且无法收回，先减仓。

### 5. 澜起科技：看存储和半导体锚

澜起科技轻亏，今天要看存储和半导体承接。

早盘处理：

- 江波龙、佰维存储、兆易创新强，澜起可观察；
- 半导体弱、澜起不能收回 270，减仓；
- 跌破 265 且无修复，按弱票处理。


| 情况 | 动作 |
|---|---|
| 亨通弱修复或继续弱 | 先减，不能等尾盘 |

只做确认动作。

如果 10:30 前核心锚没有确认，今天不要再加仓。

确认条件：

- 存储：江波龙、佰维存储持续承接；
- 海外链：中际旭创不再走弱；
- PCB：胜宏科技、沪电股份强于板块；
- 超节点：紫光、锐捷、盛科至少两个强；
